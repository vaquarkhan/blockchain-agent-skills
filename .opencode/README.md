# OpenCode Integration

OpenCode can use this repository through `AGENTS.md`, `CLAUDE.md`, and the `skills/` directory.

## Start Here

- load `skills/using-blockchain-agent-skills/SKILL.md`
- keep one matching chain preset active
- add task-specific skills only when needed

## Directory Notes

- `.opencode/skills` points to the repository skill directory
- `docs/opencode-setup.md` explains the recommended workflow
