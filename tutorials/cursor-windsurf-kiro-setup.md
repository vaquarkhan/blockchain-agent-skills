# Cursor and Windsurf Setup

- **Cursor:** install `.cursor/rules/*` via extension or `install_toolkit.py --tool cursor`.
- **Windsurf:** copy `.windsurfrules.example` and read `docs/windsurf-setup.md`.
- **Kiro:** install `.kiro/steering/*` and `docs/kiro-setup.md`.
- **OpenCode:** see `docs/opencode-setup.md` and `.opencode/skills`.
