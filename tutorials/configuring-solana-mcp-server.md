# Configuring the Solana MCP Server

1. Use `mcp/solana-rpc.mcp.json` or the server scaffold under `mcp/solana-rpc-server/`.
2. Export `SOLANA_RPC_URL` (defaults to public mainnet-beta if unset).
3. Tools: `getBalance`, `getAccountInfo`, `simulateTransaction`.
4. Pair with `skills/transaction-lifecycle` and `references/solana-program-security.md`.
