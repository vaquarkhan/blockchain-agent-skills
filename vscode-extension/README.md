# Blockchain Agent Skills for VS Code Family

Installs the blockchain skill pack into the current workspace for VS Code, Cursor, Windsurf, and VSCodium.

Download: [GitHub Releases](https://github.com/vaquarkhan/blockchain-agent-skills/releases/latest)

## Commands

- `Blockchain Skills: Install Full Toolkit`
- `Blockchain Skills: Install Core Pack`
- `Blockchain Skills: Install Agent Adapters`
- `Blockchain Skills: Install Starter Pack`
- `Blockchain Skills: Install MCP Templates`
- `Blockchain Skills: Scaffold Example`

## Local Development

1. Open `vscode-extension/` in VS Code
2. Press `F5` for Extension Development Host
3. Run commands from the palette in a test workspace

See `docs/plugin-publishing.md` and `tutorials/installing-vscode-and-jetbrains-plugins.md`.
