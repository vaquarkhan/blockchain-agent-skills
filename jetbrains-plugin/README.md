# JetBrains Plugin

Install blockchain agent skills into IntelliJ IDEA, PyCharm, WebStorm, GoLand, and DataGrip.

Download: [GitHub Releases](https://github.com/vaquarkhan/blockchain-agent-skills/releases/latest)

## Commands

- Install Full Toolkit
- Install Core Pack
- Install Agent Adapters
- Install Starter Pack
- Install MCP Templates
- Scaffold Example

## Build

```bash
cd jetbrains-plugin
./gradlew buildPlugin
```

## Local Run

```bash
./gradlew runIde
```

See `docs/jetbrains-setup.md` and `tutorials/installing-vscode-and-jetbrains-plugins.md`.
