# Examples

Four example packs showing how skills, guardrails, MCP templates, and presets combine.

| Example | Type | Best for |
| --- | --- | --- |
| `chain-provider-validation` | Runnable | Phase 2 chain adapters — local proof via pytest |
| `evm-erc20-deploy` | Runnable | EVM token deploy workflow with simulate-first |
| `solana-spl-token` | Blueprint | SPL token mint/transfer planning |
| `cosmos-ibc-transfer` | Blueprint | ICS-20 cross-chain transfer |

## Usage

1. Read `README.md` in each example
2. Open `spec.md` for requirements
3. Follow `plan.md` and `tasks.md`
4. Runnable examples: run `make smoke-test` or commands in README

Scaffold via VS Code: **Blockchain Skills: Scaffold Example**

Machine-readable flags in `registry/assets.json` under `examples`.
