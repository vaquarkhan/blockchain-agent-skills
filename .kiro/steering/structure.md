# Repository Structure

| Path | Purpose |
| --- | --- |

| `examples/` | Runnable and blueprint examples |

| `templates/` | Tx plans, readiness, incident runbooks |

| `hooks/` | Session and pre-broadcast guards |
| `agents/` | Supervisor/worker personas |
