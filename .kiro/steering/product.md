# Product Overview

This workspace uses the Blockchain Agent Skills toolkit for AI-assisted chain operations.

The goal is disciplined, simulate-first delivery across EVM, Solana, NEAR, and Cosmos families:

- plan before broadcast
- simulate every state-changing operation
- enforce guardrails for mainnet and high-value flows
- treat KMS signing, confirmation depth, and compliance as normal delivery

If a request is ambiguous, start with `skills/using-blockchain-agent-skills/SKILL.md` and classify the work before proposing on-chain actions.
